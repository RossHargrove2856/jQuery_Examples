jQuery Examples

Click

The click function bind an element to a JavaScript Event, or triggers it in the case of that event.

Hide

The hide function will hide an element, when an event triggers it.

Show

The show function will show an element, or triggers it in the case of that event.

Toggle

The toggle function bind an element to a JavaScript Event, or triggers it in the case of that event.

Slide Down

The slideDown function slides down content targeted to an element, revealing it.

This text is revealed by slideDown

Slide Up

The slideUp function slides up content tagged to an element, covering it.

This text is hidden by slideUp

Slide Toggle

The slideToggle function allows content tagged to an element to be slid down or up.

This text is hidden and revealed by slideToggle

Fade In

The fadeIn function fades content tagged to an element into sight.

Fade Out

The fadeOut function fades content tagged to an element out of sight.

Add Class

The addClass function adds a class to an element, and all of the properties associated with that class.

Before

The before function insert content in before the targeted element.

After

The after function inserts content after the targeted element.

Append

The append function inserts content between content in an element and the last element tag.

Attr

The attr function adds an ID to this text which accesses the id's class and changes the text color.

Val

The val function changes a value associated with an element.

Text

The text function changes text within an element.

Data

The dat function can store data and tag it to an element.
